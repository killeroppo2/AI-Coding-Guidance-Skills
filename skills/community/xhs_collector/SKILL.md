# 小红书爆款采集器

## 触发方式

当用户说以下任一句子时触发本技能：
- 「搜集今天的小红书爆款」
- 「采集小红书」
- 「小红书热门笔记采集」

## Quick Start

```bash
cd xhs_collector/scripts
python collector.py
```

运行后脚本会：
1. 读取 `config.json` 获取 API Token、Cookie、工作流 ID
2. 读取 `keywords.json` 获取搜索关键词列表
3. 逐个关键词调用**工作流①（搜索笔记）**获取热门笔记列表
4. 对每条笔记调用**工作流②（提取详情）**获取正文+评论+互动数据
5. 将结果格式化为 Markdown 文件，保存至 `01-内容生产/爆款参考库/` 目录

## 工作流程

```
┌─────────────┐     ┌──────────────────┐     ┌──────────────────┐     ┌────────────┐
│ keywords.json│────▶│ 工作流①：搜索笔记 │────▶│ 工作流②：提取详情 │────▶│ 生成 MD 文件│
└─────────────┘     └──────────────────┘     └──────────────────┘     └────────────┘
```

## 配置文件

- `scripts/config.json` — API Token、工作流 ID、Cookie
- `scripts/keywords.json` — 搜索关键词列表（可自定义）

## 输出格式

每次采集生成文件：`01-内容生产/爆款参考库/YYYY-MM-DD_关键词.md`

```markdown
# 小红书爆款采集 - {关键词} - {日期}

## 笔记 1: {标题}
- 作者: xxx
- 点赞: xxx | 收藏: xxx | 评论: xxx | 分享: xxx
- 发布时间: xxx
- 链接: xxx

### 正文
...

### 热门评论
1. xxx (点赞 xx)
2. xxx (点赞 xx)

---
```
