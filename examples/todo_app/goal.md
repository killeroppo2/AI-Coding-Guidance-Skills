# Goal: Build a Python Flask Todo App

Build a Python Flask todo app with REST API and SQLite storage.

## Requirements
- CRUD endpoints for todos (GET /todos, POST /todos, PUT /todos/:id, DELETE /todos/:id)
- SQLite database with a todos table
- Each todo has: id, title, completed (boolean), created_at
- Input validation on title (non-empty, max 200 chars)
- Return proper HTTP status codes
