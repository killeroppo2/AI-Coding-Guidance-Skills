# Goal: Build a CLI File Organizer

Build a CLI tool that organizes files in a directory by their extension.

## Requirements
- Accept source directory as argument
- Create subdirectories named by extension (pdf/, jpg/, txt/, etc.)
- Move files into appropriate subdirectories
- Handle files with no extension (put in "other/")
- Dry-run mode that shows what would be moved
- Summary output showing files moved per category
